rangeTo

operator fun Location.rangeTo(other: Location): CubicalShape
Content copied to clipboard

This function creates a new CubicalShape object, representing a range from one to another Location. This function is defined as a rangeTo operator function. This function uses the this as the first Location (CubicalShape.component1) and the other (CubicalShape.second) as the second Location inside the CubicalShape.

Return

a new CubicalShape object.
