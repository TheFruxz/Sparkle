fun skull(owner: String, httpRequest: Boolean = true): Item

This function creates a new Item with a Material.PLAYER_HEAD. If the httpRequest parameter is set to true, the player's skin will be fetched from the Mojang API. If the httpRequest parameter is set to false, the player's skin will be fetched from the server.
