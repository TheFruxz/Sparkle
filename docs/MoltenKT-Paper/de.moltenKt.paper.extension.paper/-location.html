This function / pseudo-constructor creates a new Location, but only provided with a worldName and the coordinates defined as general numbers. x, y and z are then converted into Doubles using Number.toDouble and yaw and pitch are converted into Floats using Number.toFloat The conversion of the numbers, are strictly tied to the toDouble and toFloat functions, so they rule the conversion processes.
